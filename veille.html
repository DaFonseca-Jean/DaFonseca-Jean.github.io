Deno

Veille technologique

Deno est un runtime pour JavaScript et TypeScript qui a pour projet de remplacer Node.

Pourquoi Deno?

Le commencement

Deno a ete lance car Ryan Dahl , le createur de Node car il avait des regrets par rapport a Node

Comment marche deno

Tout comme node deno utilise le moteur JavaScript V8 utilise dans chrome , Deno est ecrit en rust c'est un framework server side ressemblant a Node mais voulant etre meilleur sur chaque aspect (Poid , vitesse , securitee , compatibilitee..)

Exemple de projet de grande envergure

Supabase , qui est une alternative OpenSource a FireBase (Ensemble de services d'hebergements d'applications) est base sur Deno

Mon projet pour pratiquer sur Deno

Pour apprendre étant donné le peu d'informations et d'aide sur le net je me suis dit qu'il fallait pratiquer et j'ai donc commencé à faire un menu comme le Rofi Show Run pour lancer des applications sur Linux
