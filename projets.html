Trieuse d'image

Une trieuse d'image par tag (Un peu comme un Pinterest) avec trois systèmes de recherche
Par addition de tag(Images avec minimum un des tag)
Par tag principal et secondaires(Ne mettant que les images ayant ce tag en principal pour une recherche plus précise)
Par ajout de tag (Comme sur la plupart des sites , images ayant au moins les tags écrits)
Sera disponible en application , site web et Robot Discord
L'idée est de rechercher dans une base de données publique ou de se logger pour accéder à ses bases de données privées
Beta disponible sur github sous forme de bot discord (Application et site web à venir)
